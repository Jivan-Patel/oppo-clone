# OPPO Clone

## Project Description
This project is a static OPPO-inspired multi-page UI clone focused on product categories such as mobile, audio, tablets, and accessories.

## Objective
- Recreate a consumer electronics catalog layout.
- Build consistent product pages with shared styling.
- Practice multi-page navigation patterns.

## Tech Stack
- HTML5
- CSS3

## Project Structure
- `index.html`: Homepage / entry page.
- `mobile.html`: Mobile products page.
- `audio.html`: Audio products page.
- `tablets.html`: Tablets category page.
- `accessories.html`: Accessories category page.
- `styles.css`: Main/global style rules.
- `product-style.css`: Product card/category-specific styling.

## Key UI Features
- Separate category pages for major product lines.
- Product-card style presentation.
- Reusable visual styling across pages.

## How To Run
1. Open the `oppo-clone` folder in VS Code.
2. Start with `index.html` in browser.
3. Open category pages to view complete UI set.
4. Optional: Use Live Server for auto-refresh while editing.

## Customization Guide
- Edit category content directly in each HTML page.
- Tune global colors and spacing in `styles.css`.
- Modify product listing style in `product-style.css`.

## Current Scope
- Frontend-only implementation.
- No backend data source or product filters.
